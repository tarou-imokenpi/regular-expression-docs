# 正規表現一覧

|正規表現|説明|
|----|----|
|^|文字列の先頭にマッチする|
|$|文字列の末尾にマッチする|
|\b|単語境界にマッチする|
|\d|任意の数字にマッチする|
|\D|数字以外の任意の文字にマッチする|
|x?|xと言う文字に任意でマッチする(0回か1回マッチ)|
|x*|xが0回以上現れるものにマッチする|
|x+|xが1回以上現れるものにマッチする|
|x{n,m}|xがn回以上、m回以下にマッチする|
|(a|b|c)|a, b, cのどれか一つだけにマッチする|
|(x)|調査中|
